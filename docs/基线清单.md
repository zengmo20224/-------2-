## 基线详情

### v1.0.0-fb（功能基线）

- **包含的关键 CI**：CI-DOC-004 项目边界、CI-DOC-005 架构设计、CI-DOC-013 需求基线、CI-DOC-001 AGENTS.md
- **建立依据**：需求与边界经小组评审通过
- **后续变更要求**：任何功能范围变化必须走 CCB（C 类变更）

### v1.0.0-m1（里程碑基线）

- **退出条件**：H5 可稳定启动；公开数据可浏览；登录后接口可联调
- **关键变更**：修复认证请求头风险；建立可重复执行的演示种子数据
- **关键 CI**：CI-SC-001~006、CI-DB-001/002

### v1.0.0-m2（里程碑基线）

- **退出条件**：公开 H5 不依赖静态假数据；管理端发布的活动可在 H5 查看
- **关键变更**：H5 首页/服务/商品/社区/活动列表详情；营销活动后台维护
- **关键 CI**：CI-SC-003/005、CI-SC-001

### v1.0.0-m3（里程碑基线）

- **退出条件**：一次预约能从 H5 创建并在管理端正确处理
- **关键变更**：用户资料/宠物/地址；预约时段/距离校验/并发/状态流转；含 CR-20260613-001 登录方式变更
- **关键 CI**：CI-SC-001/005、CI-DB-006

### v1.0.0-m4（里程碑基线）

- **退出条件**：订单流程使用真实数据和事务规则
- **关键变更**：商品详情/购物车/下单/查询；服务端金额与库存控制
- **关键 CI**：CI-SC-001/005、CI-DB-007/008

### v1.0.0-m5（里程碑基线）

- **退出条件**：核心互动可重复验证，无私私信息泄露

## 基线管理命令

```bash
# 列出所有基线
git tag -l

# 查看某个基线对应的 commit
git show v1.0.0-m3 --stat --oneline

# 切换到某个基线查看代码
git checkout v1.0.0-m3

# 比较两个基线差异
git log v1.0.0-m2..v1.0.0-m3 --oneline

# 打新基线 tag（仅配置管理员执行）
git tag -a v1.0.0-mN -m "Milestone N: <简述>"
git push origin v1.0.0-mN
```
